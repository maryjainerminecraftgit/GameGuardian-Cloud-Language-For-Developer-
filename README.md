# GameGuardian-Cloud-Language-For-Developer-
t.me/wiqwur

To use this script, prepare language JSON files (for example, ru.json and the mandatory en.json) with text keys and upload them to GitHub in the lang/ folder, specify your GITHUB_USER, GITHUB_REPO, and GITHUB_BRANCH at the beginning of the Lua code, configure the items in the menu functions via openMenu({ {L["key"], function} }, L["title"], back_index), and run the script in GameGuardian, where it will automatically download the selected translation and be called by clicking on the text with the language selection.
